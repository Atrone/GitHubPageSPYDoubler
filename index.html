About the Project

The objective of this quantitative trading project is to develop a robust trading strategy using signal processing techniques to preprocess market data and feed it into a neural network for generating profitable trade signals. The project aims to identify patterns and trends in historical financial data, using advanced signal processing methods like wavelet transforms and Fourier analysis, to enable the neural network to make accurate predictions about future price movements. This trading strategy can then be integrated to other strategies to improve risk and/or return.

Features

  • Feature 1: Automated job to run the final optimized algorithm.
  • Feature 2: Paper trading script to test algorithm hyper parameters in different economic environments.
  • Feature 3: Real time reporting for algorithm test monitoring and live trading signals.
  • Feature 4: Windows server to support automated daily jobs for running the final algorithm and reporting the live trading signal.
  • Feature 5: AWS lambda that listens for live trading signals and executes them in a Questrade account/investment strategy.

Technologies Used

The project is built using the following technologies:

  • Technology 1: Python is the language used for the R&D of the finalized algorithm
  • Technology 2: PowerShell is the language used for adminstration of the Windows server
  • Technology 3: C++ is the language used to connect with the Questrade API
